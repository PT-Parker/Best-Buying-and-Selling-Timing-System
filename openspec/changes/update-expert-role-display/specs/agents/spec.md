## MODIFIED Requirements
### Requirement: Dynamic Expert Selection Reasoning
The system SHALL expose the highest-scoring expert rationale and display the active expert role in Chinese in the UI summary.

#### Scenario: Summary shows top-score rationale
- **WHEN** expert scoring completes,
- **THEN** the reasoning agent returns the rationale for the highest score,
- **AND** the UI shows the active role in Chinese with the rationale.
